# Notes App

A simple React + Vite notes app. Add and delete text notes with persistence via localStorage.

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Features
- Add text notes
- Delete individual notes
- Persists in localStorage across reloads
- Uses useState, useEffect, array manipulation, and component re-rendering

## Keyboard
- Enter/Control+Enter to save (when focused on input)
