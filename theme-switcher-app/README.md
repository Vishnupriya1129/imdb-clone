# Theme Switcher App

A simple React + Vite app with a global theme switcher using Context API.  
Supports Light/Dark modes, persists preference in localStorage, and includes smooth transitions.

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Features
- Global theme state via Context API
- Light/Dark mode toggle
- Persists user preference in localStorage
- Smooth transition animation
